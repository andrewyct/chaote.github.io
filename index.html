Bio

Chao-te Andrew Yang

Andrew (楊肇德) is a software developer focused on web development and back-end business logic. Currently, he is a Java developer, but he also has worked with a variety of topics and languages including system-level programming, Linux kernel hacking, mobile development, image processing with Hadoop, and cloud application deployment.

As a driven and active person, Andrew has a wide range of interests and is constantly learning new technologies. This personal website reflects his passion for technology, art, life, and photography. He holds a BS in Industrial Engineering from Yuan Ze University and an MS in Computer Science from Boston University.

  • MapReduce Focus Area Detector

    Cloud-based application for image processing

    This program detects and highlights the well-focused area of the input images. The main task of this program is CPU intensive, but the computation can be broken into multiple small tasks and can be processed more efficiently using distributed system. Built with Hadoop MapReduce framework, the program can be easily configured and deployed to a computer cluster or the cloud.

    (For full description please visit the Projects section)


  • Wretch Album Downloader

    Java online photo album downloader

    This is an online photo album downloader for www.wretch.cc (a website owned by Yahoo since 2007, but it has been closed since December 2013). The program allows user to automatically download open albums (publicly visible) of a specific account. It can be configured to manual, semi-auto, and full auto mode using different input parameters.

    In full auto mode, the program will parse the HTML information found on target account, continuously search for the next available album, and then download the images automatically until it reaches the end of the last album.

    (For full description please visit the Projects section)


  • 2012
    Interactive CPU Scheduling Simulator (2nd version)

    JavaScript web application

    The Interactive CPU Scheduling Simulator is an academic project originally designed and created by Jan Horjus and Matthew Powers-Freeling at BU. The application simulates the CPU scheduling mechanism in modern operating systems, it can be used as a tool to analyze new algorithms or to help user better understand popular ones such as FIFO, SJF, Priority Scheduling, and Round-Robin.

    In addition to the core functions in the first version, the second version has added the ability to generate random input processes, new algorithms, and statistic data to make it more informative and easier to understand.

    (For full description please visit the projects section)
