BaybioManager

Apr. 2014  â€“  

Developed by:  Chao-te Yang
Written in:  Java

BaybioManager is a desktop application designed for Parisgoshop.com, a cosmetics/organic products vendor. In addition to the official website, the company has launched several branch stores in online marketplace such as Ruten and Yahoo, each of them uses different admin panel. This application provides system admin the ability to manage transaction records and monitor inventory status in one place, without having to login to different online stores and manually check the quantities in stock. It also provide a simplfied workflow for the admin's daily operation, reducing the time required to manually insert transaction data into their ERP system. Currently the application is still under development.

From the technical point of view, the implementation of BaybioManager takes the future growth of the company into consideration and it is designed to be easy-to-maintain and easy-to-extend.

The GUI is built with a self-implemented MVC-like pattern, which separates the codes for manipulating data and displaying results. It requires extra classes to construct such design pattern, but will be easier for the developer to add new components in the future.

Data Access Objects abstracts and isolates the underlying details of database connections from the business logic, and it will come in handy when the company needs to pull data from a newly launched online store or other datasource using a different type of database. High level concurrency api like SwingWorker also makes it simple to separate the EDT and worker threads.

MySurvey is a semester long team project for the CS673 Software Engineering.  The goal is to create a web application that will provide user the ability to create, take and analyze online surveys.  Our target survey creators will be data analysts from corporations and from the education sector that are collecting data from their designated group of survey-takers.  The application will also provide pre-fabricated tools to speed up survey creation and publication.

There are many related work on the market, some are free , like SurveyMonkey which allow the user to login, create the survey, and view analytic reports, and some are costly but more sophisticated like Visimojo.  MySurvey offers a range of features geared towards both new survey creators and advanced users. This includes question templates to quickly get started in addition to customizable complex branching, time measurement for each question, and fine-tuned analytic reporting. MapReduce Focus Area Detector

Jul. 2013  â€“  Aug. 2013

Developed by:  Chao-te Yang
Written in:  Java

The Focus Area Detector is a cloud-based program capable of distinguishing the difference between a sharp line and a blurred line, the program can detect and highlight in-focus areas of the input images. Built with the Hadoop MapReduce framework, it can be easily configured and deployed to a computer cluster or the cloud. The main task of this program involves calculating the difference of the RGB value between any two consecutive pixels, it is CPU intensive, but the computation can be broken into multiple small tasks and can be processed more efficiently using distributed system.

If an object is well-focused in the picture, the details on that object should be very sharp and clear because the pixel's RGB value in that area changes rapidly.  If the object is out of focus, the details on that object will be blurred because the pixel's RGB value changes gradually from black to dark gray, from dark gray to light gray, and from light gray to white.

The MapReduce task consists of three stages: First, the Builder writes the pixel information to a text-based file. Secondly, the Mapper reads the text input, calculates the difference, and writes the output to the next stage. Finally, the Reducer calculates the total score and a new image with hightlighted area will be created.
